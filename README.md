# TN 2025 Elections Simulation

A Python-based Electronic Voting Machine (EVM) Simulator developed using Tkinter.

## Features

* EVM-style voting interface
* Voter authentication
* Duplicate vote prevention
* Vote storage using JSON
* Election result calculation
* Graphical user interface

## Technologies Used

* Python
* Tkinter
* JSON
* Matplotlib
* Seaborn

## How to Run

1. Download the project
2. Install dependencies

```bash
pip install matplotlib seaborn numpy pillow
```

3. Run the application

```bash
python main.py
```
